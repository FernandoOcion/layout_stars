Sistema de Avaliação por Estrelas

Este é um exemplo de implementação de um sistema de classificação por estrelas usando apenas HTML e CSS, conforme especificado nos requisitos.

Instruções de Layout

  • Reset do margin padrão do navegador
  • 6 blocos de estrelas com 5 elementos estrela cada
  • Modificadores stars--0 a stars--5 para controlar as estrelas ativas
  • Use background-image para as estrelas (não use <img> ou <svg>)
  • Use display: flex para o bloco de estrelas
  • Não use a propriedade gap para o flex container

Exemplo de Código HTML

<div class="stars stars--3">
    <div class="stars__star"></div>
    <div class="stars__star"></div>
    <div class="stars__star"></div>
    <div class="stars__star"></div>
    <div class="stars__star"></div>

Exemplo de Código CSS

.stars {
    display: flex;
}

.stars__star {
    width: 24px;
    height: 24px;
    background-image: url('estrela-inativa.svg');
    background-repeat: no-repeat;
    background-position: center;
    margin-right: 4px;
}

.stars--3 .stars__star:nth-child(-n+3) {
    background-image: url('estrela-ativa.svg');
}
